# Feature Ideas

Backlog of post-MVP enhancements. Promote to a design doc / plan when ready to build.

- Pitch Config selector, store known pitch dimensions for known grounds e.g Anfield, Old Trafford etc
- Per-stadium override for advertising-board height + offset (currently nominal 1.0 m / 2 m touchline / 4 m goal-line). Useful when residuals on ad-board-anchored frames are higher than expected — different stadiums use different LED ribbon heights and runoff distances.
